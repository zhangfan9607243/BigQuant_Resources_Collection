# BigQuant Resources Collection

## Overview

This repository collects selected resources from **BigQuant.com**, one of the most well-known quantitative finance development platforms in China.

**BigQuant.com** is an integrated quantitative research platform that provides:

- Financial market data 
- Cloud computing resources  
- Built-in quantitative research modules  
- Strategy development and backtesting infrastructure  

The platform enables researchers and developers to efficiently build, test, and deploy quantitative trading strategies.

## Purpose of This Repository

The primary goal of this repository is **code-focused learning**.

Unlike visual workflow tutorials, the materials here emphasize:

- Pure code-based development  
- Understanding BigQuant’s underlying modules  
- Practical implementation examples  
- Technical exploration of platform features  

This repository is designed for developers who already have programming experience and want to quickly become familiar with the tools, data structures, and computational resources available on BigQuant, specifically:

- Quantitative developers  
- Data scientists in finance  
- Algorithmic trading researchers  
- Engineers exploring Chinese quant platforms  

## Running the Code

⚠️ **Important:** All code in this repository must be executed within **BigQuant’s AIStudio environment**. Running locally or on other platforms may cause resources fail to load and results not display properly.

These scripts are not intended to run in a local Python environment, as they rely on:

- BigQuant proprietary data interfaces  
- Platform-specific modules  
- Cloud-based computing resources  

How to use the codes:

1. Create an account on BigQuant.com  
2. Open AIStudio  
3. Upload or copy the code into a notebook  
4. Execute within the platform environment
